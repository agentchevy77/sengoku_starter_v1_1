# Sengoku v0.7.0 Release Notes

- Acceptance doctrine refreshed: `ACCEPT` line now mirrors formation readiness for at-a-glance go/no-go decisions.
- Sustainment dials added so ops can monitor capital, cooldowns, and pacing pressure in one panel.
- Supply narrative lines weave the breakout/breakdown story directly into recon and command-room output.
- Recon CLI upgrades: pretty renderer, `--include-supply` enrichment, and `--mode {prob|micro}` flag for targeted intel.
- Command Room integration stitches microchips, acceptance, and supply into a unified tactical view.
- Optional alert enrichment injects supply and sustainment context into downstream notification channels.
- Health CLI exposes runtime + IBKR diagnostics (pacing alerts, cache depth) with optional `--ping` handshake.
