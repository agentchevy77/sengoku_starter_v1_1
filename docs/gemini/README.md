# Gemini CLI Artifacts

This directory stores the Gemini CLI build notebooks and reference pages (e.g. `Empire0.md`, `Empire1.md`, `Empire2.md`, etc.).

These files are required by the Gemini workflow—please keep them under version control and do not delete or rename them without coordinating with the Gemini tooling owners.

If you need to add or remove pages, document the change here so the history stays clear.
